Comandos

RECOMENDE UM FILME

Comando para recomendar

Deve ser informado em sequência deseja ouvir outra recomendação de filme.

ABRIR CALENDÁRIO

Comando utilizado para abrir calendário.

ALTERAR EXIBIÇÃO

Comando utilizado para aterar o modo de exibição do calendário.

Opções para este comando: "DIA"; "MÊS" ou "ANO"

ADICIONAR EVENTO

Comando utilizado para adicionar novo evento.

Deve ser informado o título do evento e sequência dia e mês do evento.

EDITAR EVENTO

Comando para editar eventos

Deve ser informado em sequência a data e título do evento, depois o novo título e data do evento

REMOVER EVENTO

Comando utilizado para remover eventos.

Deve ser informado dia e mês do evento, e em sequência o título do evento.

EXIBIR EVENTOS

Comando utilizado para exibir eventos.

Deve ser informado em sequência dia e mês do evento.

LIMPAR CALENDÁRIO

Comando utilizado para remover todos os eventos.

Será questionado se realmente deseja remover todos os eventos

EXIBIR LEMBRETES

Comando utilizado para exibir a pagina de lembretes

ADICIONAR LEMBRETE

Comando para adicionar lembretes

Deve ser informado em sequência o nome do lembrete

EDITAR LEMBRETE

Comando para editar lembretes

Deve ser informado em sequência o número do lembrete, deve ser iformada a nova descrição do lembrete

REMOVER LEMBRETE

Comando para remover lembretes

Deve ser informado em sequência o número do lembrete

LIMPAR LEMBRETES

Comando utilizado para remover todos lembrtes

Será questionado se realmente deseja remover todos os lembretes
