Comandos de voz

ABRIR CALENDÁRIO

Comando utilizado para abrir calendário

ALTERAR EXIBIÇÃO

Comando utilizado para aterar o modo de exibição do calendário

Opções para este comando: "DIA"; "MÊS" ou "ANO"

ADICIONAR EVENTO

Comando utilizado para adicionar novo evento

Deve ser informado em sequência dia e mês do evento

EXIBIR EVENTOS

Comando utilizado para exibir eventos

Deve ser informado em sequência dia e mês do evento
